## 0. 工程显示设置（还原 UI 的前提）

应用全局配置如下。

| 设置项 | 值 | 说明 |
|---|---|---|
| window/size/width × height | **1920 × 1080** | UI 设计分辨率，所有坐标按此布置 |
| window/size/test_width × test_height | 1280 × 720 | 编辑器测试窗口 |
| window/dpi/allow_hidpi | true | |
| window/stretch/mode | **"2d"** | 2D 拉伸（贴图按比例缩放，非视口像素拉伸） |
| window/stretch/aspect | **"keep"** | 保持 16:9，比例不符时加黑边 |
| boot_splash/image | `assets/app_icons/splash.png` | 引擎级启动图 |
| boot_splash/bg_color | (0.1098, 0.1098, 0.1098) ≈ #1C1C1C | 启动底色 |
| config/icon | `assets/app_icons/brotato_icon.png`（另有 .icns/.ico 原生图标） | 应用图标 |
| mouse_cursor/custom_image | `assets/ui/custom_cursor.png`，hotspot (3,3) | 全局自定义鼠标指针 |
| run/main_scene | `pause` | **开机第一个场景＝自制启动动画**（见 §8），播完跳 `title_screen` |

## 9. 迁移核对清单（差异提醒）

1. **字体文件**：主字体 `Anybody-Medium.ttf` 与各语言 Noto 字体位于 `assets/fonts/raw/`；应用图标 `brotato_icon.icns` / `brotato_icon.ico` 位于 `assets/app_icons/`。
2. **主题/字体样式**：九宫格、配色、字号等样式参数以上表为准，重建 UI 时按表手工配置。
3. `ui/menus/common/cross_icon..png`、`plus_icon..png` 文件名含**双点**，引用时保持原名。
4. `hud/ui_xp_bg.png`、`hud/ui_xp_fill.png`、`splash/splashart_atlas.png`、`option_button_arrow_v2.png`、`small_option_button_arrow_up.png`、`triangle_left_icon.png`、`options_icon.png`、`mods_icon.png`、`exit_icon.png`、`start_icon.png`、`resume_icons.png` 为预留资产（动态加载备用），照常放置即可。
5. `audio/sfx/ui/`、`ui/splash/*.wav`、`ui/menus/*.wav` 属音频类，见《资产对应_06》（音频文档）。
