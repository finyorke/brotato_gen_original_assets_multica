## 2. 粒子特效参数

所有效果由特效管理器统一播放，注册的效果槽位：
- 治疗粒子 → `heal_particles`
- 属性增益粒子 → `stats_boost_particles`
- 减速粒子 → `speed_removed_particles`
- 受击血粒子 → `hit_particles`
- 受击斜线特效 → `hit_effect`
- 拾取材料粒子 → `pickup_gold_particles`
- 暴击粒子 → `critical_particles`
- 一击必杀粒子（见 §2.15）

播放规则：实例池复用；`amount = max(1, amount * effect_scale)`；粒子整体朝击退反方向旋转（rotation = 方向角 − PI）；全局同屏特效上限 100。受击粒子位置 = 单位世界坐标 + 击退方向 × effect_distance。

## 3. vfx/ 全部资产

### 3.1 浮动文字（程序生成，无贴图资产）
- `floating_text`：文本标签，theme=base_theme，字体 `font_40_outline_thick`（40 号厚描边），默认尺寸约 44x45，居中对齐；附带图标精灵图层 + 缓动动画。
- `floating_text` 行为：显示时传入内容、方向、时长、散布角与颜色 → 自身染色=该颜色，位移 = 方向旋转 ±散布角/2 随机；第一段缓动：位置移动 duration 秒（弹性缓出）；第二段：缩放→0（弹性缓入缓出）+ 透明度→0（线性），再 duration 秒；结束回池。图标显示在文字右侧 +8px，缩放 (0.5,0.5)。
- 浮动文字管理器默认参数：**direction = (0,-80)，duration = 0.5，spread = PI/4**，同屏上限 100 条；带图标的金币/属性提示 duration*2，正值用全局材料绿、负值用设置项 `color_negative`（默认红），并随机播放 stat_pos/neg、harvest_pos/neg 音效。忽略的属性 = ["enemy_health","enemy_damage"]。
- `floating_text_manager` / `floating_text_manager_shop`：游戏内/商店两个变体管理器。

### 3.2 受击斜线特效 `vfx/hit_effect/`
- `hit_effect`：逐帧动画精灵图层，3 帧序列（`sprites/frame0000~0002.png`，256x256 白色斜十字斩击闪光，逐帧扩散消散），fps=20，不循环；播完即隐藏回池。
- 播放条件：受击时按 effect_scale 概率播放，位置同血粒子，随机旋转方向。

### 3.3 感叹号预警 `exclamation` + `exclamation.png`
250x250 白色感叹号贴图（染色为黄色 (1,1,0)），对象偏移 (-2,-20)。动画轨道 "animation"（1 秒，自动播放，播完自毁）：
- scale：0.5→0.8→在 0.77~0.98 间弹跳抖动（11 个关键帧，每 0.1s）；
- rotation：0°→51.4° 线性；
- self_modulate.a：0→1（0.1s）→保持→1.0s 时 0（淡出）。
用于敌人攻击预警（如冲锋前头顶感叹号）。

### 3.4 受伤暗角 `damage_vignette`
着色器为主场景内嵌资源：`COLOR = vec4(color.rgb, smoothstep(multiplier, softness, distance(UV, 0.5)))`，color=黑，softness=0.8。运行时按血量设 `multiplier = clamp(hp/max_hp, 0.3, 0.8)`：血越少 multiplier 越小、屏幕四周黑色暗角越向中心收缩（设置项 darken_screen 关闭时固定 0.8）。

### 3.5 噪声扰动着色器 `shader_noise_displacement` / `shader_noise_displacement_add`
两个画布类着色器，唯一区别：`_add` 版渲染模式为加法混合。
统一逻辑：用内置单形噪声无缝纹理（period 105.6，persistence 0.677）按 `UV + TIME*wave_speed`（再乘 wave_size 反比）采样，取灰度 `*wave_strenght - wave_strenght*0.5` 作为 UV 偏移量，再采样主贴图纹理参数，乘以染色输出。
参数：`wave_speed=1`，`wave_size=1`，`wave_strenght=0.1`，主贴图纹理。
挂载处：`jellyshield`（水母护盾果冻形变）、`ratzilla`。

### 4.2 拼接机制
- 地图为单一砖块地图层，单格 64x64（cell_quadrant_size=64），collision_layer=128。
- `ground_tiles`：0 号砖为自动砖模式（tile_mode=2），region 192x256，子砖尺寸 64x64 → 3 列 x 4 行共 12 个子砖；权重配置：子砖 (2,3)（右下角的"素面"砖）权重 50，其余 11 个子砖权重 1。
- 地面铺设规则：开局按地图宽 x 高逐格填砖，每格的子砖按权重随机抽取（权重 50/61 ≈ 82% 概率为素面砖，约 18% 为带装饰子砖）→ 形成"大面积纯色 + 稀疏装饰"的随机散布地面。**无边缘自动拼接逻辑，纯随机子砖填充。**
- 地图尺寸：`zone_data` 默认 width=32, height=24（即 2048x1536 px）；按玩家 `map_size` 加成缩放（有下限保护）。
- **换肤方式（区域换色）**：不更换图块集，仅运行时把地砖贴图替换为 tiles_1~6 之一。背景主题每局从默认背景表随机抽取（或按设置固定），**与危险等级无关**。
- 地图边框：砖块地图下挂的边框层（九宫格贴图框，texture=tiles_outline.png，patch_margin 四边 64，轴向拉伸=平铺），矩形从 (-64,-64) 覆盖 (width+2)x(height+2) 格；染色 = 当前背景的描边色。各背景描边色：
  1 DIRT (0.471,0.404,0.345)；2 FOREST (0.314,0.365,0.361)；3 VOLCANO (0.392,0.392,0.392)；4 DREAMY (0.475,0.392,0.380)；5 BONEYARD (0.345,0.306,0.259)；6 DARKLANDS (0.267,0.267,0.267)。
- 地图外部底色：独立画布层级上的背景贴图框（渐变贴图，宽 1920），第二色每局随机自 8 色暗色表（暗红/暗黄/暗青/暗蓝/暗紫/暗品红等）。
- 物理边界：地图边界逻辑在四边动态生成厚 4 格（256px）的矩形碰撞墙，非贴图。

## 5. shaders/ 三个 gdshader

### 5.1 `shaders/flash.gdshader` —— 受击闪白
- uniform：`flash_color`（默认白）、`flash_modifier`（0~1，默认 1）。
- 功能：`color.rgb = mix(color.rgb, flash_color.rgb, flash_modifier)`，整张精灵混合为纯色。
- 挂载：单位内嵌着色器材质（flash_color=白，flash_modifier=1.0），作为单位的 `flash_mat` 材质。受击时将精灵材质换成 flash_mat，闪白计时（0.1s）超时换回原材质 → **受击瞬间全身闪白 0.1 秒**。所有单位（玩家/敌人）通用。

### 5.2 `shaders/hue_shift.gdshader` —— 色相旋转
- uniform：`hue`（0~1 → 0~360° 色相旋转，绕灰轴 k=(0.577,0.577,0.577) 的 Rodrigues 旋转）、`alpha`（整体透明度乘数）、`desaturation`（0~1 去饱和，luma 权重 0.299/0.587/0.114）。
- 挂载：`hue_shift_shadermat`（hue=1.0, alpha=0.4, desaturation=0.8，resource_local_to_scene）。用于武器/单位的"幻影/虚化"显示（半透明去饱和效果）。

### 5.3 `shaders/outline.gdshader` —— 多层描边
- render_mode unshaded；uniform：`width`（0~16）、`alpha`、`desaturation`、`texture_size`、`outline_color_0~3`（最多 4 层描边色，α>0 才绘制，第 N 层采样距离 = width*N/texture_size）。
- 实现：13 方向邻域 alpha 采样累加，α>0 处用描边色与本体色按本体 alpha 混合。
- 挂载材质：
  - `weapon_outline_shadermat`：width=2，四层全透明（运行时按武器等级填色）→ 所有近战/远程武器（hand、hammer、drill……全部武器）引用。
  - `projectile_outline_shadermat`：width=3，outline_color_0=黑 → 投射物描边。
  - `ui_timeline` 内嵌材质：outline_color_0=黄(1,1,0) → 时间轴图标高亮描边。

## 6. 出生预警（怪物刷新提示）`vfx/`

| 资产 | 说明 |
|---|---|
| `vfx/entity_birth.png` | 256x256 白色粗笔 X 叉 |
| `vfx/structure_birth.png` | 256x256 白色粗笔 O 圆环 |
| `audio/sfx/spawn/birth_begin_sound.wav` / `birth_end_sound.wav` | 预警开始/实体落地音效（begin 音量 0dB、end -15dB，pitch ±0.2） |
| `entity_birth` + `entity_birth` | 触发区域（圆形半径 72，collision_mask=2 检测玩家） |

参数与行为：
- `time_before_spawn = 60`（帧，60fps 计 → 默认 1 秒预警）。
- 颜色：敌人 = 设置项 `color_negative`（默认红）每通道 +0.22 提亮；中立 = `color_positive`（默认绿）+0.22；建筑/宠物 = 矢车菊蓝，且贴图换为 O 圆环；魅惑 = 全局魅惑色。
- 闪烁时序：闪烁周期 = 剩余时间/3.25；周期到时自身染色透明度降为 0.25，持续 6 帧后回 1.0 → 越接近刷新闪烁越快（共约 3 次渐快闪烁）。
- 贴图随机旋转 0~360°；若刷新瞬间玩家压在预警点上（敌人类型），改随机位置重新计时。
