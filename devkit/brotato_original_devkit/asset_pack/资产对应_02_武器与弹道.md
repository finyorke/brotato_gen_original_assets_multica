# Brotato 资产对应表 02 —— 武器与弹道

> 元素 ID 与机制文档 03 章一致：`weapon_<名>_<tier>`（如 `weapon_knife_1`）。共 **61 把武器**（近战 36 + 远程 25）。

## 0. 通用规则（先读）

### 0.1 武器场景通用结构

每把武器都是通用武器对象的继承实例，对象分层结构固定：

```
武器（容器层）
└─ 武器贴图精灵图层 (position/offset 即“握持挂点”参数)
   ├─ 攻击判定触发区域 (碰撞层=8) → 碰撞形状 (近战攻击判定矩形)
   ├─ 出弹口挂点 (远程=出弹口，近战=粒子/参考点)
   └─ 对齐锚点挂点 (与角色武器挂点对齐，默认 (0,0))
索敌圈触发区域 (运行时按 max_range 设置)
射击行为逻辑
```

- 武器由角色的武器容器层摆位：武器位置 = 挂点位置 − 对齐锚点位置。1–6 把武器的环形布点在玩家身上（见角色章节）；>6 把时按半径 `60+(n−6)*5` 圆周均布。
- 所有武器贴图图层 **无缩放、无旋转**（scale=(1,1)、rotation=0），朝向由武器对象整体旋转实现；瞄准方向在左半边时贴图垂直翻转。
- 描边：所有武器共用 shader `shaders/outline.gdshader`，`width=2.0`，平时描边色全透明，仅高亮/诅咒时着色，无需贴图资产。

### 0.2 Tier（阶级）视觉规律 ★

**全部 61 把武器：同一武器的各阶共用同一套对象配置、同一张武器贴图、同一张 96×96 图标，不换图也不换色**。各 Tier 仅数值不同，个别武器逐阶加射程（chopper 135→180、ghost_flint/sharp_tooth 150→180、spear 350→500、fireball 400→500、flamethrower 250→400、ghost_scepter 300→450、rocket_launcher 500→600、obliterator 800→1000、wand 350→425、minigun 400→450）。阶级颜色只体现在商店/UI 的物品边框背景（UI 章节资产），战斗画面零差异。并非每把都有 4 阶（如 dextroyer/drill/excalibur/scythe/chain_gun/gatling_laser 仅 T4），见表中 Tier 列。

### 0.3 弹道通用规则

- 基类弹道对象：精灵图层＋攻击判定触发区域/碰撞形状（默认圆形 **radius=44**，有效中心对齐弹体）＋动画轨道。玩家弹道继承玩家弹道基类（碰撞层=8）。
- 未指定 `projectile_scene` 的武器（pistol、revolver、smg、minigun、chain_gun、double_barrel_shotgun）使用**默认子弹** `bullet_projectile`。
- 弹道贴图不自旋（rotation=飞行方向）；唯一例外 shuriken：`rotation_speed=25` → 每物理帧 +25°。
- 飞行至 `max_range+100`（手动瞄准 +50）后直接回收，无淡出动画。
- 弹道整体透明度乘设置项 `projectile_opacity`。
- **无独立“枪口火光”贴图**。开火反馈=后坐动画；以下武器在出弹口挂常驻粒子：torch、flaming_knuckles、fireball、wand → `torch_burning_particles`（加色混合火焰粒子，9 帧图集 `vfx/particles/particle_fire_animation.png`，amount=5、scale≈0.55）；ghost_scepter → `ghost_scepter_particles`。

### 0.4 攻击动画（运行时生成的缓动动画，无关键帧动画）★ 手感关键

近战动画由近战射击行为逻辑实时生成：

**通用时序量**
- `BASE_ATK_DURATION = 0.2s`；`atk_spd = (角色攻速% + 武器攻速修正)/100`
- `range_factor = max(0, 攻击距离 / clamp(70*(1+atk_spd/3), 70, 120))`
- 出击时长 `atk_duration = max(0.01, 0.2 − atk_spd/10) + range_factor*0.15`
- 收回时长 `back_duration = 0.2/(1+atk_spd*3)`（atk_spd≤0 时取 0.2）

**突刺 THRUST（attack_type=0）**
1. 后拉：武器贴图横向位置 − recoil（默认 25px），时长 recoil_duration（默认 0.1s），缓动曲线为指数缓出；
2. 后拉结束瞬间播放挥击音效、开启攻击判定；
3. 前刺到 x + max_range，时长 atk_duration/2（指数缓出）；
4. 关攻击判定（`deal_dmg_on_return=true` 则收回完才关，本作无武器启用），收回原位，时长 back_duration。

**横扫 SWEEP（attack_type=1）**
- 扫击距离 `atk_distance = min(max_range, max(250, 与目标距离))`；侧移量 `= atk_distance/2`；摆角 **±0.9π（162°）**；
1. 起手：position → (x−recoil, y±side)，rotation → ∓162°，时长 recoil_duration（指数缓出）；
2. 播音效、开攻击判定，线性扫到 (x + atk_distance*0.75, y)、rotation→0，时长 atk_duration/4；
3. 线性扫到另一侧 (x−recoil, y∓side)、rotation→±162°，时长 atk_duration/4；
4. 关攻击判定，指数缓出收回原位归零角，时长 back_duration。
- 起手侧依贴图当前垂直翻转状态镜像。`alternate_attack_type=true`（fighting_stick、sword、vorpal_sword、excalibur）每击在突刺/横扫间交替。

**远程后坐**：开火瞬间播音效并生成 `nb_projectiles` 颗弹（每颗方向在 ±projectile_spread 弧度内随机）；武器贴图 x − recoil，时长 recoil_duration（指数缓出），再同时长回弹；开火动画总时长 = 2×recoil_duration。

### 0.5 音效播放规则

- 统一规则：从该武器音效组 `shooting_sounds` **等概率随机**取一条全局播放；音量增益为 stats 的 `sound_db_mod`（dB，见表）；**音调随机 ±0.2**。
- 近战在“后拉完成、挥出瞬间”播；远程在出弹瞬间播。
- 武器损毁：`weapons_sounds/glass_smashable_small_break_{01–03}.wav` 随机，−15dB、pitch±0.1。
- 近战音效集缩写（文件均在 `assets/audio/sfx/weapons/`）：**S-刀挥×3** = `whoosh_weapon_knife_swing_{01–03}.wav`；**S-重挥×5** = `whoosh_swish_high_big_{01–05}.wav`；**S-快挥×4** = `whoosh_swish_high_fast_{01–04}.wav`；**S-锐挥×3** = `whoosh_swish_small_harsh_{01–03}.wav`。
