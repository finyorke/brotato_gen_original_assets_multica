**播放架构**：

- 全局播放通道（非定位音效）与 2D 定位播放通道（带世界坐标衰减的音效）各持有 **12 个播放器**、待播队列上限 **16 条**。队列满时丢弃；标记 `always_play` 的请求会挤掉队首强制入队。
- 播放参数约定：音频 + 音量增益（dB）+ 音调随机幅度 + 是否强制播放；音调 = 1.0 ± 随机幅度，均匀随机。全游戏音调随机幅度仅用 0 / 0.1 / 0.2 三档。
- 限流播放规则：按字符串 id 限流，同 id 同时播放数超过上限 `max_play` 时直接丢弃（用于 Bonk Dog 攻击、Catling Gun、Doc Moth 等连发源）。
- 武器开火音 → 见《资产对应_武器》章节；玩家脚步/受击 → 见《资产对应_玩家》章节。本表为全量清单，相关条目仍逐一列出但只作简述。
