# Brotato 道具清单（全 208 个道具）

> 本文档为道具系统设计规格，收录全部 208 个商店道具（共 209 份道具数据，其中"沙漏"含衍生道具"破损沙漏"），中文名与游戏内文案一致。
> 所有数值均为设计基准值；标注"运行时"的数值随游戏状态实时计算。

## 1. 道具数据字段说明

每个道具是一份道具数据（在物品基础数据上扩展），关键字段如下：

| 字段 | 含义 |
|---|---|
| `my_id` | 道具唯一 ID（本文表格第一列） |
| `name` | 本地化键（如 `ITEM_ACID`），对应中/英文显示名 |
| `tier` | 品质：`0=I 级(白/普通)`、`1=II 级(蓝/罕见)`、`2=III 级(紫/稀有)`、`3=IV 级(红/传奇)`（枚举中还有 DANGER_4/5，道具未使用） |
| `value` | **基础价格**。实际商店价格 = `max(1, (value + 当前波数 + value × 波数 × 0.1) × 道具价格系数 × (1 + 无尽系数))`，其中道具价格系数受 `items_price` 属性（如优惠券）影响 |
| `max_nb` | 最大持有数；`-1` 表示不限。同一玩家持有数（含商店中已出现的）达到上限后，该道具不再出现在商店 |
| `tags` | 商店标签数组，参与角色 `wanted_tags` 的商店倾向机制（见第 4 节），也用于"移除商店某类道具"等效果的匹配 |
| `tracking_text` | 物品栏悬停时额外显示的统计行（如"造成的伤害"），数据由对局内道具统计累计 |
| `unlocked_by_default` | 是否默认解锁；`false` 的道具需通过成就/挑战解锁后才会进入奖池（全部道具中有 48 个需解锁） |
| `can_be_looted` | 能否在商店/战利品中出现；仅"破损沙漏"为 `false`（只能由沙漏转化而来） |
| `replaced_by` | 购买后立即替换为另一道具（仅"沙漏"使用：购买后波次结算时替换为"破损沙漏"） |
| `effects` | 效果条目数组；普通属性用基础效果条目（key+value），特殊机制用专门的特殊效果类型 |
| `is_cursed` / `curse_factor` | 诅咒物相关（DLC 机制，基础道具均为 `false` / `0.0`） |
| `is_lockable` | 能否在商店中锁定（全部道具均为 `true`） |

### 回收价

回收所得 = `min(商店价, max(1, 商店价 × clamp(0.25 + recycling_gains/100, 0.01, 1.0)))`，基础为 25%，"回收装置"+35% 后为 60%。`value=1` 的道具（破损沙漏）回收固定得 1。

### 道具品质出现概率

商店每个栏位先按波数/幸运掷出品质（品质抽取公式）：每档品质有 `MIN_WAVE`（最低波数）、基础概率、每波加成概率与上限，幸运按 `波数加成 × (1 + 幸运/100)` 放大。再从该品质奖池随机抽取道具。

## 2. 道具与角色的联动：`wanted_tags` 商店倾向机制

商店道具栏位的倾向规则：

- 每个商店道具栏位有 **5%** 概率（标签倾向率 = 0.05；若有 `boosted_wanted_item_tag` 效果则为 **10%**）触发"标签倾向"：该栏位**只保留**与当前角色 `wanted_tags` 有交集的道具（即按 `item.tags` 过滤）。
- 角色 `banned_items` / `banned_item_groups` 中的道具会被从奖池移除（最后一波之后即无尽模式不再移除，改用无尽禁用表）。
- 武器栏位另有独立机制：20%（前 12 波最高 +5%）概率只出现已持有的同名武器、35%（前 12 波最高 +20%）概率只出现与已持有武器同类（武器套装即刀刃/钝器/枪械等**武器职业**——套装数据定义的是武器职业套装加成，**道具本身没有套装系统**）。

### 4.3 沙漏 / 破损沙漏（唯一的 replaced_by 道具）

- **沙漏 Hourglass**（IV 级，100 价格，max_nb=1）：持有后离开商店时 `current_wave += 1 - 沙漏数`，即**当前波数倒退 1**（等于重打上一波）；副作用：下一波以 1 HP 开始。
- 结算后道具自动移除，并按 `replaced_by` 自动替换为**破损沙漏 Broken Hourglass**（value=1、max_nb=0、不可购买、无任何属性，仅保留"下一波 1 HP 开始"的描述文本与占位）。
- 商店"前往下一波"按钮文本会实时显示倒退后的波数。
